# Tv-shows
## Introduction
Application is used to finding and save Tv shows. It can view photos and descriptions. The app also has features such as display, create, edit, and delete users lists, mark them as favorites, and add TV shows to these lists. You can return to the TV show details from the saved lists. It is designed to work both on desktop computers and mobile devices. This application was made using: Angular 4.1.1, Angular cli 1.0.0 , TypeScript 2.3.2, HTML, CSS/SCSS, Bootstrap 4.0.0-alpha.6, REST.
